Volunteer Management System

Welcome to the Volunteer Management System. This application helps you manage volunteer registrations, track attendance, and generate PDF reports.

To get started:

  1. Install Node.js and MongoDB on your system
  2. Open terminal/command prompt in this folder
  3. Run: npm install
  4. Start MongoDB service
  5. Run: npm start
  6. Open http://localhost:3000
Start Application

Features:

  • ✅ Add new volunteers with complete information
  • ✅ Auto-fill existing volunteer data
  • ✅ Generate PDF attendance sheets
  • ✅ Search and manage volunteer records
  • ✅ MongoDB integration for data storage
